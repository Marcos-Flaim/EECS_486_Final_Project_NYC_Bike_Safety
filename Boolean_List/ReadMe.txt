Use ranked intersections file to create a boolean list of properties
for each intersecion (eg. has pedestrian ramp, has bad pavement etc.)
Used in embeddings recomendation to determine best intervention.

To run:
Ensure Master_Top_500_Intersections.csv is in the data folder
Install required packages (csv, requests, json, haversine)
run program

*Note: this program takes a long time to run ~2hrs
